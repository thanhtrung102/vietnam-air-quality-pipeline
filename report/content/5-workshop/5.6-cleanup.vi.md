Mọi thứ đều được quản lý bởi Terraform, nên việc gỡ bỏ chỉ là một lệnh duy nhất.

```bash
cd terraform
terraform destroy        # removes all ~82 resources
```

## Lưu ý
- **Các S3 bucket** có bật versioning có thể cần được làm trống các object trước nếu `force_destroy` không được đặt:
  ```bash
  aws s3 rm s3://openaq-pipeline-<suffix> --recursive
  ```
- **Bucket chứa remote state** (`openaq-tfstate-thanhtrung102`) được khởi tạo ngoài luồng (out-of-band) và *không*
  bị xóa bởi `terraform destroy` — chỉ xóa thủ công nếu bạn ngừng vận hành hoàn toàn.
- **ECR image** (`openaq-forecast-generate`) và object nguồn của **CodeBuild** vẫn tồn tại nếu được tạo
  bên ngoài lần apply chính; xóa chúng thủ công nếu muốn.
- Chi phí sau khi gỡ bỏ trở về ~$0; các vỏ storage/catalog/state tốn vài cent cho đến khi được xóa.

## Xác minh việc gỡ bỏ
```bash
aws lambda list-functions --query 'Functions[?starts_with(FunctionName,`openaq`)].FunctionName'   # empty
terraform plan    # should propose re-creating everything (confirms state is clean)
```
